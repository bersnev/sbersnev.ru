Команды Git

Установка

    sudo apt-get update

    sudo apt-get install git

Первоначальные настройки

    git config --global user.name "Bersnev Sergey" - имя пользователя

    git config --global user.email bersnev@tut.by - e-mail

    git config --global core.editor "code --wait" - установка редактора

    git config --list - проверка настроек общая

Общие команды

    git init - инициализация

    git status - список незакомиченных файлов

    git add . - добавление к будущему комиту

    git commit -m "комментарий к комиту" - коммит

    git push или git push origin master - пуш коммита в удаленный репозиторий

    git fetch - скачать изменения из удаленного репозитория

    git pull - скачать изменения из удаленного репозитория и смержить

    git checkout -B master origin/master - Привести ветку в состояние удаленного репозитория !!! Осторожно, удалит все локальные коммиты, которых нет в удаленной ветке

    git branch new-branch - создание ветки

    git checkout new-branch - переключение на ветку

    git branch -d branch-name - удаление ветки

    git branch -m old_name new_name - переименование ветки

    git branch -v - просмотр веток
