Получение информации

    man - выводит руководство пользователя (справку) по команде

    pwd - выводит сведения о директории в которой находишься

    ls - выводит содержимое директории

    ps - позволяет просматривать сведения о работающих процессах

    clear - очистить консоль

    grep word -r project - найти слово word во всех файлах в папке project

    find . -iname '*ind*' - найти в текущей папке (и подпапках) все файлы, имена которых содержат ind и показать списком

    ls -a >> file.txt - записать в file.txt результат вывода команды ls -a

    chmod +x ./fileName - сделать файл исполняемым

    mydir=$(pwd) - извлечь информацию из вывода команды и назначать её переменной

    echo $mydir - вывод на экран переменной

Манипуляции с файловой системой

    cd - изменение рабочей директории

    cd . - текущая директория

    cd .. - переход к родительской папке

    touch - создание файла

    mkdir - создание директории

    cp - копирование файла

    mv index.html old - перемещение файла в папку

    mv order.txt orderNew.txt - переименовать файл

    rm ghost.png - удалить файл

    rm -rf old - удалить папку и всё из нее

    ln - создание ссылки

bash команды для управления

    ; - выполнение команды одна за другой

    && - команда справа не будет выполняться, если команда слева не выполнена успешно

    & - свернуть подобный процесс в фон, используйте & в конце команды. Тогда процесс выполняется в фоне, не блокируя ввод в окно терминала

    | - передает результат выполненной инструкции следующему пайпу

    ls -l | grep .md$ | less - command1 | command2 | command3

    || - Двойной пайп (ИЛИ) - вторая команды будет выполнена только в случае НЕУДАЧНОГО заверения первой команды
